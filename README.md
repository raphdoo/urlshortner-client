### Installation
 
First, you need to clone the repository into your local machine
```javascript
$ git clone https://github.com/raphdoo/urlshortner-client.git
```

Then, you need to install the dependencies.
```javascript
$ npm install
``` 

Finally, you need to run the server.
```java
$ npm run start
```
